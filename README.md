# Random Password Generator
### Oasis Infobyte - Python Programming Internship

## About
A secure Random Password Generator built with 
Python and Tkinter GUI.

## Features
- Adjustable password length (4 to 64 characters)
- Toggle uppercase, lowercase, digits, symbols
- Exclude specific characters
- Generate 1, 3, 5 or 10 passwords at once
- Real-time password strength meter
- One-click clipboard copy
- Session history window

## How to Run
```bash
# Optional - for clipboard support on Linux
pip install pyperclip

python password_generator.py
```

## Tech Stack
- Python 3
- Tkinter (GUI)
- secrets module (secure randomization)
```
